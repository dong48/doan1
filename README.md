# Shopee Consumer Insights Dashboard (Streamlit)

## 1. Giới thiệu
Dashboard hỗ trợ phân tích hành vi tiêu dùng trên dữ liệu Shopee, cung cấp các biểu đồ trực quan và bộ lọc tương tác
nhằm rút ra insight theo ngành hàng và theo phân khúc người bán.

## 2. Link sản phẩm
- Demo (Streamlit): https://shopee-consumer-insights.streamlit.app/
## 3. Công nghệ sử dụng
- Python
- Streamlit
- Pandas, NumPy
- Matplotlib / Plotly 
- Squarify (TreeMap)


## 4. Hướng dẫn chạy
### Bước 1 
pip install -r requirements.txt
### Bước 2
streamlit run shopee_streamlit.py
